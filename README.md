# 포근한 걱정 보관소
> 걱정을 인형에게 털어놓고, 바람에 날려보내거나 벽난로에 태워버리세요.
<br>
어떤 고민들은 차분히 생각하면서 어딘가에 적어내려가는 것만으로도 한결 나아집니다. <br>
'포근한 걱정 보관소'는 일상에서 겪는 막연한 불안과 걱정을 디지털 공간에서 마주할 수 있게 합니다. <br>
그리고 작성한 내용을 종이비행기로 날리거나 벽난로에 태우는 모션을 통해 심리적 환기를 돕습니다. <br>

<br>

## 주요 기능
**걱정인형**: 인형을 클릭하여 걱정을 입력하고 털어놓을 수 있습니다.

**감정 해소 모션**
- 종이비행기 모션: 고민을 적은 종이를 창문을 통해 하늘로 날려 보냅니다.
- 벽난로 모션: 고민을 적은 종이를 벽난로에서 태워 없앱니다.

**데이터 관리 시스템**
- 저장 스위치: 스위치의 상태에 따라 고민을 로컬 스토리지에 저장하거나, 세션 종료 시 자동으로 삭제합니다.
- 기록 열람: 책장을 클릭하면 그동안 모아둔 고민 리스트를 조회할 수 있습니다.
- 초기화: 휴지통을 클릭하여 모든 기록을 삭제합니다.
